--- layout: post author: RaeEE454 excerpt_separator: --- A step-by-step guide to creating a C++ file within the Linux terminal.

Picture this: You've been using a code editor to create and run your projects. The code editor? Visual Studio Code. However, you want a challenge. Maybe something not as visually appealing, no effort into GUI whatsoever. That's where the terminal comes in. The terminal has a command line where you sit in the cockpit of your computer. It can be daunting for newcomers who are used to an app or website editor. But like you said before, you want a challenge.

This tutorial will step you through the process of creating a C++ file within the Linux terminal. This is not designed to teach the C++ language itself; rather, it is designed to apply a basic understanding of C++ and your terminal.


At-A-Glance C++ Facts

Fact #1:
C++ is one of the most widely-used programming languages.

Fact #2:
Your computer understands 0s and 1s, known as binary. To translate your code into something your computer understands, you will need a compiler which for Linux is GNU. Learn more about compilers here.

Fact #3:
You can use C++ to gain more control of system resources and memory.

Fact #4:
You'll come across an executable file. This is a sharable file compatible across systems.

Let's Get Started!

Step 1:
Create a new directory. You may title it however you like.


Step 2:
Navigate to directory.


Step 3:
Make sure you have GNU Compiler Collection (GCC) installed. You will need this to run your C++ code. To install, click here.


Step 4:
Confirm GCC is installed successfully.


Step 5:
Create your file and insert your code. Here is a VIM cheatsheet to help you navigate.

You will see a blank screen. Press a key such as "a" to start writing.

Save by pressing esc and :w. Exit by pressing esc and :x.


Step 6:
Once you have finished with your code, compile your file. Nothing will appear after you press enter.


Step 7:
You can list your executable files with the command $ ls -ltr.


Step 8:
You can run your file by entering the command $ ./a.out.



There You Have It!


You have successfully created a C++ file in your Linux terminal! You are now ready to conquer the world with your new found skills!
